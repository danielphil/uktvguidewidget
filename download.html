Changelog

Version 1.3

  • Channels are now sorted in alphabetical order. Note that if you are upgrading the widget, you'll have to reload the channel list before this change will take effect. This can be done by switching to the preferences side of the widget and clicking the Reload Channel List button.
  • Fixed an issue where characters in listings occasionally became corrupted. If you spot any more occurrences of this, please let me know!
  • Tested and confirmed that the widget is working on Snow Leopard.
  • Updated widget URLs to reflect moving to the new Google App Engine hosted site at http://tvguidewidget.appspot.com.

Version 1.2

  • Mouse wheel and trackpad scrolling can now be used with the listings.
  • Fixed an issue where the date selector would not update properly when the current date changed.
  • Fixed an issue where the channel list would not update properly after modifying the list of channels in preferences.
  • Please note: This version of the widget now requires MacOS X 10.4.3 (Tiger) or above.

Version 1.1

  • Fixed an issue that stopped with widget working with Leopard.
  • The widget is now a universal binary.
  • Added a check for new version button.

Version 1.0

  • First release of the widget.
